- 프로젝트 환경설정
	-빌드하고 실행하기
콘솔로 이동
1. ./gradlew build
2. cd build/libs
3. java -jar hello-spring-0.0.1-SNAPSHOT.jar
4. 실행 확인

윈도우 사용자를 위한 팁
1.콘솔로 이동 명령 프롬프트(cmd)로 이동
2. ./gradlew gradlew.bat 를 실행하면 됩니다.
3. 명령 프롬프트에서 gradlew.bat 를 실행하려면 gradlew 하고 엔터를 치면 됩니다.
4. gradlew build
5. 폴더 목록 확인 ls dir
6. 윈도우에서 Git bash 터미널 사용하기
